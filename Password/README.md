# Password Generator

## Project Description

Password Generator is a simple Python-based application that helps users create random and secure passwords.

This project allows users to enter the required password length and generates a password containing letters, numbers, and special characters.

I created this project to understand Python GUI development and how random values can be used to generate passwords.


## Features

- Generate random passwords
- User can select password length
- Includes uppercase and lowercase letters
- Includes numbers (0-9)
- Includes special characters (!, @, #, $, etc.)
- Simple and easy-to-use interface
- Clear button to remove input and output


## Technology Used

- Python
- Tkinter Library
- Random Module
- String Module
